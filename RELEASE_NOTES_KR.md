# Release Notes (기술 버전 - 국문) - v1.8.2 (2026-03-20 03:41 UTC)

## 🏗️ 시스템 자동화 및 Gmail 안정성 강화
- **Docker 이미지 자동 정리**: VPS 서버의 디스크 효율성 및 성능 최적화를 위해 불필요한 Docker 이미지를 매일 자동으로 정리(`docker image prune -a`)하도록 자동화했습니다.
- **Gmail 스캔 커서 연동 수정**: 스캔 성공 후 마지막 메시지 ID(Cursor)가 정상적으로 저장되지 않던 문제를 해결하여, 스캔 중복이나 데이터 누락을 방지했습니다.
- **Gmail 로직 리팩토링**: 불필요한 리턴 타입을 제거하고 함수 구조를 단순화하여 Gmail 스캔 및 저장 품질을 개선했습니다.
- **다중 형식 릴리즈 노트 관리**: 기술/독자층별로 3개의 파일로 분리하고 타임스탬프 기반 이력 관리를 도입했습니다.
- **Gmail 수신인 필터링 정교화**: `To`, `CC`, `BCC` 헤더를 구분하여 사용자가 메일의 직접 수신인(`To:`)인 경우에만 업무로 할당하는 로직을 최종 적용했습니다.
- **재귀적 아이덴티티 탐지**: 별도 설정 없이도 사용자 이름과 이메일 주소를 기반으로 본인 여부를 자동 판별하도록 개선했습니다.
- **프론트엔드 상태 동기화**: JS 모듈 캐시 이슈로 인해 대시보드 업무 수치가 실제 리스트와 일치하지 않던 현상을 수정했습니다.
